# 设计文档 / Design Documents

## 中文

本目录保存监控平台功能设计和实施计划。`superpowers/specs/` 记录需求与设计
决策，`superpowers/plans/` 记录相应的实施步骤。文档可能描述历史阶段；判断
当前行为时应以现有源码和测试为准。

## English

This directory contains feature specifications and implementation plans for the
monitoring platform. `superpowers/specs/` records requirements and design
decisions, while `superpowers/plans/` records implementation steps. Documents
may describe historical stages; current source and tests define actual behavior.
