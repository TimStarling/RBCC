# 界面设计资源 / Dashboard Design Assets

## 中文

本目录保存监控界面的可编辑设计源文件。`monitor-page-source.svg` 是界面布局
与视觉元素的源稿；运行时使用的 1920×1080 PNG 位于上一级目录。修改 SVG 后
应按原比例导出 PNG，并在目标显示器上检查中文字体、坐标沙盘和状态覆盖层。

## English

This directory stores editable dashboard design assets.
`monitor-page-source.svg` is the source layout; the runtime 1920×1080 PNG is
stored one level above. After editing the SVG, export a proportional PNG and
verify Chinese fonts, the coordinate sandbox, and dynamic overlays on the
target display.
