# 噪声模块设计文档 / Noise Module Design Documents

## 中文

本目录保存噪声分析与工业警告音功能的设计说明和实施计划。
`superpowers/specs/` 是设计依据，`superpowers/plans/` 是实施记录。当前参数
和行为请以源码、`config.py` 与自动测试为准。

## English

This directory stores design specifications and implementation plans for noise
analysis and industrial warning audio. Treat the current source, `config.py`,
and automated tests as the authority for runtime behavior and parameters.
