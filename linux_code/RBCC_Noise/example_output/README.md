# 示例输出 / Example Output

## 中文

本目录提供一次噪声分析的示例结果：频谱图、频率/功率 CSV、JSON 分析结果和
生成的 WAV 警告音。文件仅用于理解输出格式，不代表任何现场的安全阈值或认证
结果。

## English

This directory contains one example analysis result: a spectrum image,
frequency/power CSV, JSON analysis, and synthesized WAV alarm. These files
demonstrate the output format; they are not safety thresholds or certified
results for a real site.
