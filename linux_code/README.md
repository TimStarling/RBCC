# RBCC Linux 软件 / RBCC Linux Software

## 中文

本目录保存运行在 Orange Pi/Linux 主机上的应用与离线安装包。

| 目录 | 功能 |
| --- | --- |
| `RBCC_Interface/` | 1920×1080 监控界面，汇总串口、TCP、视觉与噪声状态 |
| `RBCC_Camera/` | 基于 OpenCV 的工作区、重物与人员检测 |
| `RBCC_Noise/` | 环境噪声分析、自适应警告频率选择和警告音生成 |
| `media_packages_20260720/` | Ubuntu 22.04 ARM64 离线多媒体依赖包 |

`RBCC环境监测.desktop` 用于桌面启动。各模块包含独立安装与运行说明。

## English

This directory contains applications and offline packages for the Orange
Pi/Linux host.

| Directory | Purpose |
| --- | --- |
| `RBCC_Interface/` | 1920×1080 dashboard combining serial, TCP, vision, and noise state |
| `RBCC_Camera/` | OpenCV workspace, load, and worker detection |
| `RBCC_Noise/` | Noise analysis, adaptive warning-frequency selection, and alarm synthesis |
| `media_packages_20260720/` | Offline Ubuntu 22.04 ARM64 multimedia packages |

`RBCC环境监测.desktop` is the desktop launcher. See each module README for
installation and operation details.
