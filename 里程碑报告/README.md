# 项目里程碑资料 / Project Milestone Materials

## 中文

本目录保存三个阶段的 PDF 报告和一段 MP4 演示视频：

- `里程碑一.pdf`
- `里程碑二.pdf`
- `里程碑三.pdf`
- `演示视频.mp4`

这些文件用于展示项目在对应阶段的设计与结果，可能与当前源码存在迭代差异。
复现实行行为时应以仓库最新源码、配置和模块 README 为准。GitHub 网页可预览
PDF；视频建议下载后使用常见 MP4 播放器观看。

## English

This directory contains three PDF milestone reports and one MP4 demonstration
video. They document the design and results at specific project stages and may
differ from the latest source. Use the current code, configuration, and module
READMEs when reproducing runtime behavior. GitHub can preview the PDFs; download
the MP4 for playback if browser preview is unavailable.
