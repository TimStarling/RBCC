# RBCC 前端设置 / RBCC Frontend Settings

## 中文

本目录是前端设置与迁移资料的占位目录。当前没有可维护的设置文件；本地遗留的
空目录和依赖目录不纳入版本控制。后续如增加环境变量示例、部署参数或操作说明，
请避免写入真实密码、令牌和 API 密钥。

## English

This is a placeholder for frontend configuration and migration notes. No
maintainable settings are currently supplied; empty local directories and
dependency folders are excluded from version control. Future environment
examples and deployment parameters must not contain real passwords, tokens, or
API keys.
