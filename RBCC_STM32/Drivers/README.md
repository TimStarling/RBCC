# STM32 驱动依赖 / STM32 Driver Dependencies

## 中文

本目录包含 ST 提供的 CMSIS 与 STM32F4 HAL 驱动，是 CubeMX 工程依赖。除非
正在进行明确的供应商库升级，不应直接修改这些文件。许可证见各子目录中的
`LICENSE.txt`；升级时应由 CubeMX/STM32Cube 包统一更新并重新全量编译。

## English

This directory contains ST CMSIS and STM32F4 HAL dependencies generated with
the CubeMX project. Do not edit them directly except as part of an intentional
vendor-library upgrade. See each subtree's `LICENSE.txt`. Upgrade through the
matching STM32Cube package and perform a full rebuild afterward.
